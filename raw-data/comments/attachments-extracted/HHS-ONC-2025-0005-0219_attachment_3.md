AI Transcription agent
Perceives: clinical conversations
Thinks: identifies relevant clinical information
Acts: creates structured documentation

Research agent
Perceives: patient details
Considers: links to relevant specialist literature
Acts: provides evidence- based recommendations

Care coordination agent
Perceives: patient plans and needs
Thinks: identifies gaps in care and priorities
Acts: schedule appointments and follow up appointments

Scout agents: discover and retrieve relevant information from diverse sources, just as
scout ants find food sources
Worker agents: process and transform data into useful formats, similar to worker ants
processing resources
Guardian agents: ensure security, privacy, and compliance like soldier ants protecting
the colony
Coordination(s) agents: Orchestrate activities and manage resources, similar to how
queen ants coordinate colony activities
Multi agent orchestration
Orchestration layer: coordinates agent activities, manage workflows, and ensures
coherent execution across the network
Agent communication: standardized protocols allow agents to share information,
request services, and coordinate actions (MCP, A2A…)
Task decomposition: complex workflows are broken down into specialized subtasks
assigned to appropriate agents
Governance and control: policies, permissions, and oversight mechanisms ensure
agents operate within appropriate boundaries
Three pillars the agentic web
Open data platform

• Breaks down data silos with vendor-neutral, standardized formats
• Creates a single, unified patient record across all systems
• Enables real-time analytics and seamless integration through open APIs

Agent networks

• Uses specialized AI agents for clearly defined tasks
• Solves problems through collaborative and autonomous workflows
• Continuously learns and improves with each use

AI assistants

• Provides a natural language interface for every interaction
• Understands intent and context for proactive assistance
• Customizes to personal preferences and workflows
